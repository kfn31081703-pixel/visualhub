## 🏠 PUBLIC HOME (사용자용 웹사이트)

### 페이지 구성

#### 1. 랜딩 페이지 (`/`)
**목적**: 첫 인상, 브랜딩, SEO

**섹션**:
1. **Hero Section**
   - 캐치프레이즈: "AI가 만드는 무한한 웹툰 세계"
   - CTA 버튼: "웹툰 보러가기"
   - 배경: 애니메이션 효과

2. **Featured Webtoons** (인기 웹툰)
   - 슬라이더/캐러셀
   - 썸네일 + 제목 + 장르 + 조회수
   - 클릭 → 웹툰 상세 페이지

3. **How It Works** (AI 자동 생성 과정)
   - 인포그래픽
   - 키워드 → 시나리오 → 이미지 → 웹툰 완성
   - 애니메이션 효과

4. **Stats Section** (통계)
   - 생성된 웹툰 수
   - 총 에피소드 수
   - 조회수 총합
   - 지원 언어 수

5. **Latest Episodes** (최신 에피소드)
   - 그리드 레이아웃
   - 썸네일 + 제목 + 장르 + 날짜

6. **Footer**
   - 링크: About, Contact, Terms, Privacy
   - 소셜 미디어 아이콘
   - 언어 선택

#### 2. 웹툰 갤러리 (`/gallery`)
**목적**: 모든 웹툰 탐색

**기능**:
- **필터**:
  - 장르 (액션, 로맨스, 판타지, 코미디 등)
  - 언어 (한국어, 영어, 일본어 등)
  - 정렬 (최신순, 인기순, 조회수순)
  
- **검색**:
  - 제목, 키워드 검색
  - 자동완성

- **그리드 레이아웃**:
  - 카드 형식: 썸네일 + 제목 + 장르 + 조회수 + 평점
  - Hover 효과: 미리보기 이미지

- **무한 스크롤** 또는 **페이지네이션**

#### 3. 웹툰 상세 페이지 (`/webtoon/{project_id}`)
**목적**: 특정 웹툰의 모든 에피소드 보기

**구성**:
1. **헤더**:
   - 웹툰 제목
   - 장르 태그
   - 평균 평점 (★★★★☆)
   - 총 조회수
   - 구독 버튼 (선택)

2. **에피소드 목록**:
   - 에피소드 번호 + 제목 + 썸네일 + 날짜
   - 클릭 → 웹툰 뷰어

3. **설명/줄거리**:
   - 웹툰 소개
   - 캐릭터 소개

4. **댓글 섹션**:
   - 사용자 댓글 (선택적 기능)
   - 평점 시스템

#### 4. 웹툰 뷰어 (`/webtoon/{project_id}/episode/{episode_id}`)
**목적**: 웹툰 읽기 (세로 스크롤)

**기능**:
- **세로 스크롤** (웹툰 표준)
- **이미지 로딩 최적화** (Lazy Loading)
- **네비게이션**:
  - 이전 에피소드 / 다음 에피소드 버튼
  - 목록으로 돌아가기
- **전체화면 모드**
- **모바일 최적화**:
  - 터치 스크롤
  - 핀치 줌 (확대/축소)
- **댓글/평점** (하단)
- **공유 버튼** (SNS)

#### 5. 검색 페이지 (`/search?q={keyword}`)
**목적**: 키워드 검색 결과

**기능**:
- 검색 결과 그리드
- 필터 옵션
- 연관 검색어 제안

## 🔐 ADMIN DASHBOARD (관리자용)

### 페이지 구성

#### 1. 로그인 페이지 (`/admin/login`)
**기능**:
- 이메일 + 비밀번호
- JWT 토큰 인증
- Remember Me

#### 2. 대시보드 홈 (`/admin`)
**목적**: 한눈에 보는 통계

**위젯**:
1. **생성 통계** (오늘/이번 주/이번 달)
   - 생성된 에피소드 수
   - 진행 중인 작업 수
   - 성공률 (%)
   - 실패한 작업 수

2. **Job 현황** (실시간)
   - 대기 중: 5개
   - 실행 중: 3개
   - 완료: 120개
   - 실패: 2개

3. **최근 활동**
   - 타임라인 형식
   - 에피소드 생성 완료, Job 실패 등

4. **인기 웹툰 TOP 5**
   - 조회수 기준

5. **수익 현황** (선택)
   - 플랫폼별 수익
   - ROI

6. **비용 추적** (AI API 사용량)
   - Text Engine 비용
   - Image Engine 비용
   - 총 비용

#### 3. 프로젝트 관리 (`/admin/projects`)
**목적**: 웹툰 프로젝트 전체 관리

**기능**:
- **프로젝트 목록**:
  - 테이블 형식
  - 제목 + 장르 + 에피소드 수 + 상태 + 생성일
  - 검색/필터

- **프로젝트 생성 버튼** (+)
  - 모달: 제목, 장르, 키워드, 톤, 세계관 입력
  - 폼 검증

- **프로젝트 상세** (클릭 시)
  - 프로젝트 정보 편집
  - 에피소드 목록
  - 캐릭터 목록
  - 통계 (총 조회수, 평균 평점)

#### 4. 에피소드 관리 (`/admin/projects/{id}/episodes`)
**목적**: 특정 프로젝트의 에피소드 관리

**기능**:
- **에피소드 목록**:
  - 에피소드 번호 + 제목 + 상태 + 생성일
  - 상태: draft, queued, running, done, failed

- **에피소드 생성 버튼** (+)
  - 모달:
    - 에피소드 번호 (자동 증가)
    - 제목 (선택)
    - 시놉시스 (선택)
  
- **에피소드 상세** (클릭 시):
  - 시나리오 텍스트 (읽기 전용)
  - 스토리보드 (패널 목록)
  - 생성된 이미지 (패널 미리보기)
  - 최종 웹툰 미리보기
  - Job 히스토리 (어떤 작업이 실행되었는지)

- **🚀 1클릭 웹툰 자동 생성 버튼**:
  - 버튼: "웹툰 자동 생성" (Generate Webtoon)
  - 클릭 시 모달:
    - 키워드 입력 (태그 형식)
    - 목표 단어 수 (1000-3000)
    - 목표 패널 수 (10-20)
  - 확인 → API 호출 → Job 생성 → 진행 상황 실시간 추적

#### 5. Job 모니터링 (`/admin/jobs`)
**목적**: 모든 작업 추적

**기능**:
- **Job 목록**:
  - 테이블: ID + 타입 + 상태 + 생성일 + 완료일 + 비용
  - 필터: 타입, 상태
  - 검색: Job ID

- **실시간 업데이트**:
  - WebSocket 또는 폴링
  - 상태 변경 시 자동 갱신

- **Job 상세** (클릭 시):
  - 입력 데이터 (JSON)
  - 출력 결과 (JSON)
  - 에러 메시지 (있을 경우)
  - 재시도 버튼 (실패 시)

- **진행률 표시**:
  - 프로그레스 바
  - 단계별 상태: Text → Director → Image → Lettering → Packaging

#### 6. 캐릭터 관리 (`/admin/projects/{id}/characters`)
**목적**: 프로젝트별 캐릭터 관리

**기능**:
- **캐릭터 목록**:
  - 이름 + 설명 + 참조 이미지

- **캐릭터 생성 버튼** (+)
  - 모달:
    - 이름
    - 설명
    - 외형 (appearance)
    - 성격 (personality)
    - 참조 이미지 업로드
    - 스타일 프리셋 선택

- **캐릭터 편집**:
  - 정보 수정
  - 참조 이미지 추가/삭제

#### 7. 웹툰 미리보기 (`/admin/preview/{episode_id}`)
**목적**: 생성된 웹툰 확인

**기능**:
- 웹툰 뷰어와 동일
- 다운로드 버튼
- 다시 생성 버튼 (재생성)

#### 8. SNS 배포 관리 (`/admin/sns`)
**목적**: SNS 예약 업로드 관리

**기능**:
- **채널 목록**:
  - 플랫폼 + 채널명 + 상태
  - 연결/해제

- **예약 발행 목록**:
  - 에피소드 + 채널 + 예약 시간 + 상태

- **새 예약 생성** (+):
  - 에피소드 선택
  - 채널 선택
  - 예약 시간 선택
  - 메타데이터 (제목, 해시태그, 설명)

- **즉시 발행 버튼**:
  - 테스트용

#### 9. 번역 관리 (`/admin/translations`)
**목적**: 다국어 번역 관리

**기능**:
- **언어 목록**:
  - 한국어, 영어, 일본어, 중국어 등

- **에피소드별 번역 상태**:
  - 원본 (ko) + 번역본 (en, ja, zh)
  - 번역 실행 버튼

- **번역 미리보기**:
  - 원본 vs 번역본 비교

#### 10. 통계 및 분석 (`/admin/analytics`)
**목적**: 성과 추적

**기능**:
- **차트**:
  - 일별 생성 에피소드 수
  - 플랫폼별 조회수
  - 장르별 인기도
  - 국가별 조회수

- **테이블**:
  - 에피소드별 상세 통계
  - 비용 vs 수익

## 🛠️ 기술 스택

### Frontend
- **프레임워크**: Next.js 14 (App Router)
- **언어**: TypeScript
- **스타일링**: Tailwind CSS
- **상태 관리**: Zustand 또는 Redux Toolkit
- **데이터 Fetching**: TanStack Query (React Query)
- **폼**: React Hook Form
- **차트**: Chart.js + react-chartjs-2
- **애니메이션**: Framer Motion
- **아이콘**: Heroicons, Lucide React

### 인증
- **JWT**: JSON Web Token
- **저장**: localStorage (토큰)

### 배포
- **호스팅**: Vercel (Next.js 최적화)
- **API**: https://toonverse.store/api
